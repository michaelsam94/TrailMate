# WalkPlanner — Google Play Store Assets

All graphics required for a Play Console listing.

## App icon

| File | Size | Use |
|------|------|-----|
| `app-icon-512.png` | 512 × 512 | Play Console **App icon** (required) |

Launcher icons are also updated in `app/src/main/res/mipmap-*` and adaptive vectors in `drawable/ic_launcher_*.xml`.

## Featured graphic

| File | Size | Use |
|------|------|-----|
| `feature-graphic-1024x500.png` | 1024 × 500 | Play Console **Feature graphic** (required) |

## Phone screenshots

Folder: `phone-screenshots/` — 1080 × 1920 (9:16)

Upload at least **2** (max 8):

1. `01-home-plan-route.png` — Home / plan a route
2. `02-route-suggestions.png` — Route suggestions with map
3. `03-active-session.png` — Active tracking session
4. `04-generated-routes.png` — Generated route options
5. `05-route-options.png` — Additional route view

## 7-inch tablet screenshots

Folder: `7inch-tablet-screenshots/` — 1200 × 1920

Same screens as phone, scaled for 7" tablets.

## 10-inch tablet screenshots

Folder: `10inch-tablet-screenshots/` — 1600 × 2560

Same screens as phone, scaled for 10" tablets.
